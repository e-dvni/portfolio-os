# Portfolio OS

A Mac-style desktop portfolio built with React and a real window manager.

## Features
- Desktop + Dock UI
- Draggable / resizable windows
- Spotlight search
- Finder-style project browser
- Interactive Admin Dashboard (demo)
- Full-stack architecture (React + Rails API)

## Tech Stack
- Frontend: React (Vite, TypeScript)
- Backend: Rails API (coming next)
- Database: PostgreSQL (coming next)
